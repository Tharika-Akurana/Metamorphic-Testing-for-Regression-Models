\chapter{Timeline}

The project has been divided into 18 sequential milestones, each of which corresponds to the planned project activities, in order to guarantee its effective completion within the allotted time period of 32 weeks (two semesters).

\vspace{0.5cm}
1.  Selection of the Project
\begin{itemize}
  \item Study the software testing and metamorphic testing project areas that are available.
  \item Select a suitable and feasible project topic.
  \item Obtain supervisor approval for the selected project.
\end{itemize}

\vspace{0.5cm}
2. Literature Review and Regression Model Analysis
\begin{itemize}
  \item Review the literature in-depth on the oracle problem and metamorphic testing.
  \item Examine current regression models and the difficulties associated in testing them.
  \item Determine research gaps and provide evidence for the importance of the suggested solution.
\end{itemize}

\vspace{0.5cm}
3. Submission of Project Proposal Report for Supervisor’s Comments
\begin{itemize}
  \item Create the project proposal report's first draft.
  \item Send the proposal to the supervisor for review and comments.
  \item Review and document supervisor feedback for improvements.
\end{itemize}

\vspace{6 cm}
4. Submission of Final Project Proposal Report
\begin{itemize}
  \item Revise the proposal report based on supervisor feedback.
  \item Finalize project objectives, methodology, and timeline.
  \item Submit the completed project proposal report.
\end{itemize}

\vspace{0.5cm}
5. Project Proposal Presentation and VIVA
\begin{itemize}
  \item Create presentation materials that provide an overview of the project proposal.
  \item Deliver the project proposal presentation.
  \item Attend the proposal VIVA and respond to evaluation questions.
\end{itemize}

\vspace{0.5cm}
6. Identification of Metamorphic Properties and Relations
\begin{itemize}
  \item Identify suitable metamorphic properties for regression models.
  \item Based on expected outputs and input transformations, define metamorphic relations.
  \item Validate identified relations through theoretical analysis.
\end{itemize}

\vspace{0.5cm}
7. Design of Generalized Metamorphic Testing Template
\begin{itemize}
  \item Create a template for universal and reusable metamorphic testing.
  \item Describe the processes for creating, executing, and validating tests.
  \item Make sure it can be expanded to accommodate other regression models.
\end{itemize}

\vspace{0.5cm}
8. Proof of Concept (PoC) Development
\begin{itemize}
  \item Conduct a proof of concept to show that the suggested strategy is feasible.
  \item Use sample regression models to validate important metamorphic linkages.
  \item Examine PoC results and make any necessary design adjustments.
\end{itemize}

\vspace{0.5cm}
9. Development of Python-Based Metamorphic Testing Framework
\begin{itemize}
  \item Develop the core Python-based metamorphic testing framework.
  \item Implement modules for the creation and execution of automated test cases.
  \item Integrate metamorphic relations into the framework.
\end{itemize}

\vspace{3cm}
10. Project Progress Presentation and VIVA
\begin{itemize}
  \item Prepare progress presentation highlighting completed work.
  \item Present implementation progress to the evaluation panel.
  \item Attend the progress VIVA and address feedback.
\end{itemize}

\vspace{0.5cm}
11. Submission of Project Progress Report
\begin{itemize}
  \item Prepare a detailed progress report documenting completed milestones.
  \item Include implementation details and preliminary results.
  \item Submit the progress report for assessment.
\end{itemize}

\vspace{0.5cm}
12. Experimental Evaluation
\begin{itemize}
  \item Create test configurations to assess the framework's efficacy.
  \item Execute experiments using regression models.
  \item Evaluate the performance and capability of fault detection by analyzing the data.
\end{itemize}

\vspace{0.5cm}
13. Web-Based Platform Development
\begin{itemize}
  \item Design and implement a web-based platform for the testing framework.
  \item Enable user interaction for test execution and result visualization.
  \item Integrate the web platform with the backend testing framework.
\end{itemize}

\vspace{0.5cm}
14. Documentation and Final Writing
\begin{itemize}
  \item Prepare comprehensive documentation of system design and implementation.
  \item Write chapters of the final project report.
  \item Verify adherence to the formatting requirements of the university.
\end{itemize}

\vspace{0.5cm}
15. Submission of Draft Final Report for Supervisor’s Comments
\begin{itemize}
  \item Submit the draft final report to the supervisor for review.
  \item Collect and analyze feedpback.
  \item Identify required revisions.
\end{itemize}

\vspace{0.5cm}
16. Submission of Final Project Report
\begin{itemize}
  \item Revise the final report based on supervisor feedback.
  \item Complete the document by reviewing it.
  \item Submit the completed final report.
\end{itemize}

\vspace{0.5cm}
17. Final Presentation and VIVA
\begin{itemize}
  \item Prepare final presentation summarizing the entire project.
  \item Deliver the final presentation.
  \item Attend the final VIVA and defend the project outcomes.
\end{itemize}

\vspace{0.5cm}
18. Final Demonstration
\begin{itemize}
  \item Demonstrate the complete metamorphic testing system.
  \item Display the characteristics of the web-based platform and experimental findings.
  \item Conclude the project with final evaluation and submission.
\end{itemize}
